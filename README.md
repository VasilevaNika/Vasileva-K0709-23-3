# Проект: Dating Telegram Bot

Учебный dating-сервис в виде бота: пользователь оформляет профиль, листает чужие анкеты, ставит лайки или пропускает карточки. При взаимном интересе появляется мэтч и возможность переписки. Порядок показа анкет строится на многоуровневом рейтинге (статический по данным профиля, динамический по реакциям других людей и итоговая смесь).

## Что будет в продукте

- Вход и идентификация по данным Telegram (без отдельного пароля).
- Профиль с полями вроде имени, возраста, пола, города, текста «о себе», тегов интересов и галереей фото.
- Лента кандидатов с действиями «лайк» и «пас».
- Мэтч при совпадении лайков и чат в рамках пары.

## Технологии (план)

Backend и бот на **Python**; к Telegram подключаемся через **aiogram** или **python-telegram-bot**. Основная БД — **PostgreSQL**. Для ускорения выдачи ленты предусмотрен **Redis**. Файлы фото можно выносить в **S3-совместимое хранилище** (например **MinIO**). **Celery** и **брокер очередей** (RabbitMQ, Kafka и т.п.) — по мере необходимости для фоновых задач и событий.
